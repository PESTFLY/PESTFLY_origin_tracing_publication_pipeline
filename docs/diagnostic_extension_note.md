# Note on diagnostic SNP-panel development

The reduced diagnostic SNP-panel workflow is not included in this public manuscript pipeline.

Formerly developed as Step 6 and Step 6b, that workflow should be kept as a separate private/developmental extension. In the future diagnostic-development pipeline, the public steps should retain the same numbering:

```text
00-07  identical to this published origin-tracing workflow
08     ranked diagnostic SNP resources
09     empirical minimal SNP-panel benchmarking
```

This separation prevents confusion between the genome-wide origin-tracing evidence used in the manuscript and the downstream translational work aimed at simplified laboratory assays.
