# Publication release checklist

Before making this repository public:

- [ ] Confirm all co-authors agree on code release.
- [ ] Add the final GitHub URL to `CITATION.cff`.
- [ ] Replace placeholder author fields in `CITATION.cff`.
- [ ] Choose and add the final license.
- [ ] Confirm whether raw input data can be redistributed.
- [ ] If raw data cannot be redistributed, provide instructions for obtaining or reconstructing inputs.
- [ ] Run `Rscript run_publication_pipeline.R` from a clean checkout.
- [ ] Confirm the final report is produced in `results/07_authority_report/`.
- [ ] Tag the release as `v1.0-manuscript-origin-tracing` or equivalent.
