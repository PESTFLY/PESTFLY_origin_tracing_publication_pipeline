# Results folders

The default result folders match the public step numbering.

```text
results/00_fasta/
results/01_qc/
results/02_snp_panels/
results/04_origin_assignment/
results/05_loo_validation/
results/06_rf_corroboration/
results/07_authority_report/
```

Step 03 writes mutual-information annotations into `results/02_snp_panels/` because it enriches the Step 02 SNP-panel resources.
